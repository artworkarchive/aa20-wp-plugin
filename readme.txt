Artwork Archive wordpress plugin which lets you embed your public profile portfolio into a WordPress page.

== Description ==

For Artwork Archive users with a Public Profile page, this allows you to embed your public portfolio into a wordpress page.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create a new WordPress Page or edit an existing page to add the "short link" to embed the portfolio

== Frequently Asked Questions ==

= Who does this work for =

Anyone with an Artwork Archive account who has a Public Profile setup.
